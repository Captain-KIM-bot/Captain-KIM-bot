파이썬 변수와 자료형

기초부터 예제까지 차근차근 알아보기

🔸 변수란?

변수는 데이터를 저장하기 위한 이름표입니다.
파이썬에서는 별도의 자료형 선언 없이 변수에 값을 할당할 수 있습니다. 

# 변수 선언 예시
name = "홍길동"
age = 25
height = 180.5
is_student = True

🔸 주요 자료형

  • int : 정수형
  • float : 실수형
  • str : 문자열
  • bool : 불린형 (참/거짓)
  • list : 리스트
  • dict : 딕셔너리

🔸 자료형 예제

# 정수형
a = 10

# 실수형
b = 3.14

# 문자열
c = "안녕하세요"

# 불린형
d = True

# 리스트
e = [1, 2, 3, 4]

# 딕셔너리
f = {"이름": "철수", "나이": 30}

🔸 자료형 확인

type() 함수를 사용하면 변수의 자료형을 확인할 수 있습니다.

print(type(a))  # 
print(type(c))  # 
print(type(e))  #

💡 팁

  • 변수명은 영문자, 숫자, 밑줄(_)로 구성되며 숫자로 시작할 수 없습니다.
  • 파이썬의 예약어(ex. if, while, for 등)는 변수명으로 사용할 수 없습니다.
